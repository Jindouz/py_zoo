
# Overview
A Python program for managing a dictionary list of animals in a zoo.

## Start the program
Open zoo.py to start the program (or ```py zoo.py``` in VSC terminal) 

## Features
* Display all animals
* Add animal by Name/Species/Age 
* Delete by ID 
* Search by ID
* Save changes to a JSON file on Exit
